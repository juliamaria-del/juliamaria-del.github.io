Seção 2 — Ranking de Tecnologias Relevantes

  1. JavaScript

    Linguagem essencial para interatividade no lado do cliente e cada vez mais forte no servidor.

  2. Python

    Muito utilizada em Ciência de Dados e no desenvolvimento ágil de backends robustos.

  3. SQL

    Linguagem padrão para gerenciamento e consulta de bancos de dados relacionais.

  4. Docker

    Tecnologia de containers que facilita a criação e o deploy de ambientes de desenvolvimento.

  5. TypeScript

    Superset do JS que adiciona tipagem estática, reduzindo erros em grandes projetos.

Seção 3 — Glossário Técnico

HTML
Linguagem de marcação utilizada para estruturar o conteúdo da Web.
Abreviação de HyperText Markup Language.
URL
Endereço Web
Caminho que identifica a localização de um recurso disponível na rede.
CSS
Linguagem de estilos usada para definir a apresentação visual de páginas HTML.
DOM
Interface de programação para documentos HTML e XML, representando a estrutura da página como uma árvore de objetos.
API
Conjunto de definições e protocolos que permite que diferentes softwares se comuniquem.
HTTP
Protocolo de transferência de hipertexto para comunicação cliente/servidor.
Protocolo que opera na camada de aplicação do modelo OSI.
JavaScript
JS
Linguagem de programação interpretada de alto nível que permite implementar funcionalidades complexas em páginas web.
DNS
Sistema que traduz nomes de domínio amigáveis (como google.com) em endereços IP.
